Посібник Користувача: 3D Векторна Система

Цей посібник допоможе вам зрозуміти та ефективно використовувати інтерактивну 3D векторну систему, створену за допомогою A-Frame.

1. Загальний огляд

Додаток дозволяє візуалізувати двовимірні вектори на площині XY, виконувати базові векторні операції (додавання, віднімання, скалярний та векторний добуток) та спостерігати за їх геометричною інтерпретацією в 3D просторі.

2. Інтерфейс Користувача (UI)

Всі елементи керування розташовані у верхній частині екрану.

2.1. Вектори A та B

  • Поля вводу (X, Y): Ви можете ввести значення X та Y для Вектора A (червоний) та Вектора B (синій). Підтримуються десяткові дроби (наприклад, 3.5).
  • Кнопка "Оновити A" / "Оновити B": Після введення координат натисніть відповідну кнопку, щоб оновити візуалізацію вектора на сцені.

2.2. Векторні операції

  • "A + B (Паралелограм)": Обчислює суму векторів A і B. На сцені відображається результуючий вектор (зелений) та допоміжні лінії (помаранчеві) для побудови паралелограма.
  • "A - B (Трикутник)": Обчислює різницю векторів A і B. На сцені відображається результуючий вектор (зелений) та допоміжна лінія (помаранчева), що ілюструє віднімання векторів методом трикутника.
  • "A · B (Скалярний)": Обчислює скалярний добуток векторів A і B. Результат (скалярне число) відображається в панелі результатів, а також показується кут між векторами.
  • "A × B (Векторний)": Обчислює векторний добуток векторів A і B. У 2D це скалярне значення, яке також відображається в панелі результатів.
  • "Очистити": Приховує всі результати операцій та допоміжні лінії зі сцени.

2.3. Приклади

Ці кнопки дозволяють швидко завантажити попередньо визначені пари векторів для демонстрації:

  • A(3,2) B(1,4)
  • A(-2,3) B(4,-1)
  • A(5,0) B(0,3)
  • A(2,2) B(-2,2)

3. Інформація та керування сценою

На панелі керування ви знайдете важливу інформацію щодо навігації та координат:

  • Початок координат: (0,0) - Центр координатної сітки.
  • ЛКМ (затиснути) + рух миші: Обертання сцени.
  • Колесо миші: Збільшення/зменшення (масштабування) сцени.
  • ПКМ (затиснути) + рух миші: Панорамування (переміщення) сцени.

4. Візуалізація на сцені

  • Координатна сітка: Тло сцени відображає сітку для зручності орієнтації.
  • Осі X (червона) та Y (синя): Візуалізація осей координат з числовими позначками.
  • Вектор A: Червона лінія зі стрілкою та міткою "A".
  • Вектор B: Синя лінія зі стрілкою та міткою "B".
  • Результуючий Вектор: Зелена лінія зі стрілкою та міткою, що вказує на результат операції (наприклад, "A+B").
  • Допоміжні лінії: Помаранчеві лінії, які використовуються для візуалізації побудов паралелограма або трикутника. Зверніть увагу: ці лінії є суцільними для кращої видимості.
  • Індикатор кута: Жовта дуга з текстовою міткою, що відображає кут між векторами після скалярного добутку.
Примітка: Початок координат та інструкції з керування сценою завжди видимі в UI оверлеї для вашої зручності.
